About CryptoMan

Your modern, fast, and simple cryptocurrency trading platform.

What is CryptoMan?

CryptoMan is a user-friendly digital cryptocurrency trading platform that allows users to effortlessly buy and sell crypto at the push of a button. It integrates a live API for the latest crypto prices — updated every 60 seconds — and provides powerful tools to help users monitor, manage, and analyze their crypto portfolio.

Key Features

  • Buy & Sell Crypto – Purchase by specifying the dollar amount, no need to calculate exact shares.
  • Transaction History – Track all transactions, both open and closed.
  • Open Positions – View live data on current holdings, profit/loss, and price movement.
  • Top 20 Cryptocurrencies – Get updated prices for the most popular cryptocurrencies globally.
  • Clean UI – Intuitive and visually organized dashboard for a smooth user experience.

Technologies Used

CryptoMan is built using a modern Java tech stack:

  • Java 17
  • Spring Boot & Spring MVC
  • Spring Data JPA (migrating to JdbcTemplate)
  • Thymeleaf for server-side rendering
  • MariaDB & SQL
  • Gradle for build automation
  • JUnit & Mockito for testing
